Hello World


11 y.o.

I fell in love with programming at a very early age -- when it was introduced to me in computer class during third grade. While we only had computer class twice a week, I always looked forward to those days. The first project I had ever published was a Number Guessing Game that I worked on in my free time in sixth grade.

i.
13 y.o.

I was always fascinated by how video games could "react" to players in ways that movies could not, and, at the time, I was obsessed with the game Pokemon. So I resolved to teach myself how to make games. I began by reading the official Java 8 Reference Book, and for the longest time, I couldn't wrap my mind around Object Oriented Programming.

ii.
High School

By now, I've had many small successes, and even more failures in programming, each of them reinforcing my love for the discipline. Until now, due to a lack in programming maturity, my projects always started in full sprint, only to end in an unorganized disaster.

Fueled by an intense passion for programming and video games, I started, yet again. This time, planning far ahead. I had finally developed and released my first game Kingdom Dash for iOS and Android.

iii.
University of California, Berkeley

Now, after programming for nearly a decade with multiple projects under my belt, I'm turning my sights to research and academia. I was the primary author of MIMOSAS, a machine learning and data science software suite for nuclear security analysis, that my research group and I released just this fall.

iv.

Castle Blitz

Java, OpenGL, iOS, Android

Castle Blitz is a turn-based online multiplayer strategy game that I developed from scratch in Java. I have always been fascinated by game engine development -- it's a delicate balance of complexity and performance coupled with artistic expression.

Polyture

Java Script, React, Python

Polyture is an interactive big data analytics platform designed to speed up the data processing pipeline and allow data scientists to "touch" their data. I'm currently working on this at the Berkeley Skydeck startup incubator.

Kingdom Dash

Java, OpenGL, iOS, Android

Kingdom Dash is a game built on top of a 2D game engine that I designed and developed. It includes custom animation, physics simulation, and other gameplay features. The game engine is cross-platform, and can compile to any desktop (Linux, macOS, Windows) or mobile (iOS, Android) platform.

MIMOSAS

Python

MIMOSAS is an end-to-end modular data processing and machine learning analytics suite of algorithms developed for Nuclear Security applications. I was the primary author of this software package at the Complexity Group at the University of California, Berkeley.

Nimbleats

Java Script, React Native

Nimbleats is a mobile food ordering platform that I co-founded, designed, developed, and raised a pre-seed round for. At its peak, we had 9 restaurants on our platform in UC Berkeley's surrounding area. Though now defunct, my partner and I learned many lessons from its failure.
